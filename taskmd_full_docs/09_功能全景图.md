# 功能全景图（Feature Panorama）

## Layer A：核心任务系统（Core Task System）
- Markdown 主文件
- Stable ID
- Metadata
- CRUD
- Direct Editing Compatibility

## Layer B：日常视图（Daily Views）
- list / today / next / overdue / stats / risk
- tags / filter / search / archive

## Layer C：终端体验（Terminal UX）
- Rich Dashboard
- Live Reload
- Section Progress
- Themes
- Heatmap

## Layer D：时间与专注（Time & Focus）
- Auto-Timestamp
- Recurring Tasks
- Focus Mode
- Quick Capture

## Layer E：导出与共享（Export & Sharing）
- ICS
- PDF
- CSV / JSON
- Image
- HTML Board
- Weekly Report

## Layer F：Easy Start / Context Actions
- Task Sidecar
- `tm open <id>`
- `tm start <id>`
- resource opening / context-aware start

## Layer G：Academic Mode
- Course Highlighting
- Deadline Import (confirmed)
- Material Sync
- Risk / Weight View

## Layer H：TODO Harvesting
- scan TODO / FIXME / NOTE
- review and confirm import

## Layer I：Remote Storage
- synced folders
- SSH / SFTP backend

## Layer J：AI Layer
- task grooming
- prioritization suggestions
- report generation
- natural language commands

## Layer K：Connectors / Ecosystem
- maps / commute / weather
- finance / price
- course portals
- meeting systems
- plugin hooks
