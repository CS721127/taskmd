# License Note

This file is a **license recommendation note** for the initial repository setup.

## Recommended default

For a developer-facing CLI tool like TaskMD, a practical default is:

**MIT License**

## Why MIT is a good fit

- simple and permissive
- easy for users and contributors to understand
- common for CLI tools and developer productivity projects

## Alternatives

- **Apache-2.0** — if you want a more explicit patent grant
- **GPLv3** — if you want strong copyleft requirements

## Recommendation

Unless you have a strong reason otherwise, start with **MIT** to reduce adoption friction.

## Important

Before public release, replace this note with the actual chosen license text in `LICENSE`.
